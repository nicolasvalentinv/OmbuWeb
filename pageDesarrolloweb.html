Desarrollo Web Front End

Full Stack: Encargado de manejar cada aspecto relacionado con la creación y mantenimiento de una aplicación web. FrontEnd: Es la parte de una aplicación web que se encarga de interactuar con los usuarios, por ello decimos que está del lado del cliente. Son todas las tecnología de diseño y desarrollo web que corren en el navegador y que se encargan de la interactividad con los usuarios, algunos ejemplos de estos tenemos: HTML, CSS, JavaScript, React, Angular, Bootstrap, entre otros.
