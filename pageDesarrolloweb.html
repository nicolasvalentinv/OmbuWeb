HTML Introducción

Introducción a HTML y Explicación de Etiquetas Importantes HTML (Hypertext Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Permite estructurar y presentar el contenido de una página web, lo que incluye texto, imágenes, enlaces y otros elementos multimedia. Para lograr esto, HTML utiliza una serie de etiquetas o "tags" que definen la estructura y el significado del contenido en una página web.

En este momento aprenderemos como usar nuestras etiquetas más comunes dentro de lo que es el HTML, empezando por como comenzar nuestro proyecto en Visual Studio Code.

<body>

el término "body" se refiere al elemento <body> que se encuentra en el lenguaje de marcado HTML (Hypertext Markup Language). El elemento <body> es una parte fundamental de una página web y se utiliza para definir el contenido principal visible de la página que los usuarios ven en su navegador web.

<header>

La etiqueta <header> se utiliza para definir la cabecera o encabezado de una sección o página web. Esta sección generalmente contiene elementos como el logotipo del sitio web, el título principal de la página, enlaces de navegación y otra información relevante para la parte superior de la página. El contenido dentro de <header> suele ser consistente en toda la página o en una sección específica del sitio.

<ol> - <ul> - y <li>

en HTML, las etiquetas <ol>, <ul>, y <li> se utilizan para crear listas ordenadas y no ordenadas, y para definir los elementos individuales de la lista.

La etiqueta <li> se utiliza para definir cada elemento individual en una lista, ya sea ordenada o no ordenada. Debe estar contenida dentro de una etiqueta <ol> o <ul> para indicar qué lista pertenece.

<ol></ol> (Ordered List - Lista Ordenada): La etiqueta <ol> se utiliza para crear listas ordenadas, donde los elementos de la lista se enumeran automáticamente con números o letras. Cada elemento de la lista se define utilizando la etiqueta <li>.

<ul></ul> (Unordered List - Lista No Ordenada): La etiqueta <ul> se utiliza para crear listas no ordenadas, donde los elementos de la lista no se numeran y generalmente se presentan con viñetas (puntos, círculos, cuadrados, etc.). Al igual que con las listas ordenadas, cada elemento de la lista se define utilizando la etiqueta <li>..

ETIQUETA (PASO 4) <ol> - <ul> - y <li>

Las listas en HTML se utilizan para organizar y presentar información de una manera estructurada y ordenada. Hay tres tipos principales de listas en HTML: <ol> - <ul> - y <li>

<h1>, <h2>, <h3>

Las etiquetas <h1>, <h2>, <h3>, y así sucesivamente hasta <h6>, se utilizan en HTML para definir los encabezados o títulos de una página web. Cada una de estas etiquetas representa un nivel de encabezado diferente, donde <h1> es el encabezado principal y <h6> es el encabezado de nivel más bajo.

ETIQUETA (PASO 5) <h1>, <h2>, <h3>

Esta etiqueta se utilizan en HTML para definir los encabezados o títulos de una página web. Cada una de estas etiquetas representa un nivel de encabezado diferente <h1>, <h2>, <h3>

<section>

La etiqueta <section> se utiliza para dividir el contenido de una página web en secciones temáticas o áreas distintas. Cada sección puede tener su propio encabezado (<header>), contenido y pie de página (<footer>). La etiqueta <section> ayuda a organizar y estructurar el contenido de manera semántica, lo que facilita la comprensión de la página y su navegación.

ETIQUETA (PASO 5) <section>

<p>

La etiqueta <p> en HTML se utiliza para definir párrafos de texto en una página web. Sirve para estructurar y formatear el texto de manera que se separe en bloques legibles y coherentes. Cada vez que desees presentar un nuevo párrafo de texto en tu página web, deberías utilizar la etiqueta <p>.

<img> - <a>

La etiqueta <img> en HTML se utiliza para insertar imágenes en una página web. Esta etiqueta permite a los desarrolladores web incorporar gráficos, fotografías, iconos y otros tipos de imágenes en el contenido de una página web.

La etiqueta <a> en HTML se utiliza para crear enlaces o hipervínculos a otras páginas web, archivos, ubicaciones dentro de la misma página, direcciones de correo electrónico y más. La etiqueta <a> permite a los usuarios hacer clic en el texto o la imagen marcada con esta etiqueta para acceder al recurso vinculado.
